# App Lux 🚀

Desarrollo enfocado en la estructuración de lógica de negocio y maquetación avanzada con JavaScript. Implementación de una interfaz fluida con manejo de estados dinámicos y optimización de rendimiento frontend.

## 🔗 Demo Interactiva
¡Podés ver y probar la aplicación web armada y funcionando en tiempo real haciendo clic acá!:
👉 **[Ver Aplicación en Vivo](https://micaelaresolani.github.io/app-lux/)**

---

## 🛠️ Tecnologías Utilizadas
* **JavaScript (ES6+)** - Lógica del sistema y manejo de estados dinámicos.
* **HTML5 & CSS3** - Arquitectura de maquetación avanzada y animaciones fluidas.
* **Manipulación avanzada del DOM** - Interfaz de usuario interactiva y responsiva.

## 📈 Características del Proyecto
* Interfaz fluida y moderna orientada a la experiencia de usuario.
* Estructura de código limpio y modularización.
* Optimización de rendimiento frontend.
